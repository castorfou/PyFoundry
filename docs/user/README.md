!!! info "À propos de cette section"
    Ces pages contiennet des guides pour les **utilisateurs** du template PyFoundry.

    Elle explique **comment créer** un projet issu du template, et ensuite **comment faire vivre** ce projet, sans nécessiter de connaissances approfondies du code source.

## Contenu de ces guides

Voici les types de guides disponibles :

### Guide de [démarrage](getting_started.md)
- Prérequis système
- Création d'un nouveau projet
- Liaison pour un projet pré-existant

### Guide d'[utilisation](usage.md)
- Mise à jour vers la dernière version du template (cruft)
- Mkdocs : lancement, navigation, formats
- Modification env python (uv)
- Installation applications dans devcontainer (apt)
- Gitlab : CI/CD et CLI
- Docker
- Autres fonctionnalités
- Options du template : gpu, docker-in-docker, claude code

### Guide de [dépannage](troubleshooting.md)
- Problèmes courants et solutions

### [Conventions](conventions.md)
- Structure du projet
- Imports python et tests
- Qualité de code
- Documentation
